# Web Scraping Project

## Project Title
Web Scraping of Books Data from Public Website

## Objective
The objective of this project is to collect data from a public website using Python web scraping techniques. The extracted data is stored in a structured format for further analysis and visualization.

## Tools & Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas
- Jupyter Notebook / VS Code

## Project Description
This project extracts information from a website and stores it in a CSV file. The collected data includes:

- Product/Book Name
- Price
- Rating
- Availability Status

The data is cleaned and organized using Pandas for future analysis.

## Project Workflow

1. Send a request to the target website.
2. Retrieve HTML content.
3. Parse HTML using BeautifulSoup.
4. Extract required information.
5. Store data in a Pandas DataFrame.
6. Export the data into a CSV file.

This project demonstrates how web scraping can be used to collect real-world data from websites efficiently. The extracted data can be utilized for data analysis, visualization, and business insights.
